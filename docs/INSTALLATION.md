# Installation Guide

## Option 1: Upload ZIP in Shopify Admin
1. In Shopify Admin, go to **Online Store → Themes**.
2. Select **Add theme → Upload zip file**.
3. Upload `release/unifyone-modern-commerce-theme.zip`.
4. Preview the theme and publish when ready.

## Option 2: Shopify CLI
1. Install Shopify CLI.
2. Authenticate and link your store.
3. Push the theme from source:
   - `shopify theme push`

## Compatibility
- Built for Shopify Online Store 2.0 themes.
- Recommended to use latest Shopify storefront features.
